== Zero-extend word (32-bit)

Mnemonic::
zext.w _rd_, _rs_

Encoding::
[wavedrom]
....
{reg:[
    { bits:  7, name: 0x3b, attr: ['OP-32'] },
    { bits:  5, name: 'rd' },
    { bits:  3, name: 0x0, attr: ['ZEXT.W/ADD.UW']},
    { bits:  5, name: 'rs' },
    { bits:  5, name: 0x00 },
    { bits:  7, name: 0x04 },
]}
....

Description::
Zero-extend the least-significant 32 bits of _rs_ to XLEN and place in _rd_.

Operation::
[source,sail]
--
X(rd) = EXTZ(X(rs)[31..0]);
--
