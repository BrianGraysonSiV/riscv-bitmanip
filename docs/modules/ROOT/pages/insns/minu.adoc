== Minimum, unsigned

Mnemonic::
minu _rd_, _rs1_, _rs2_

Encoding::
[wavedrom]
....
{reg:[
    { bits:  7, name: 0x33, attr: ['OP'] },
    { bits:  5, name: 'rd' },
    { bits:  3, name: 0x7, attr: ['MINU']},
    { bits:  5, name: 'rs1' },
    { bits:  5, name: 'rs2' },
    { bits:  7, name: 0x05, attr: ['MIN/MINU'] },
]}
....

Description::
Places the smaller of two unsigned integers (_rs1_, _rs2_) in _rD_.

Operation::
[source,sail]
--
let rs1_val = X(rs1);
let rs2_val = X(rs2);

let result = if   rs1_val <_u rs2_val
    	     then rs1_val
	     else rs2_val;

X(rd) = result;
--
