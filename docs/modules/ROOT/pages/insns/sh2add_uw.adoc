== Shift unsigned word left by 2 and add

Mnemonic::
sh2add.uw _rd_, _rIndex_, _rBase_

Encoding::
[wavedrom]
....
{reg:[
    { bits:  7, name: 0x3b, attr: ['OP-32'] },
    { bits:  5, name: 'rd' },
    { bits:  3, name: 0x4, attr: ['SH2ADD.UW'] },
    { bits:  5, name: 'rs1' },
    { bits:  5, name: 'rs2' },
    { bits:  7, name: 0x10, attr: ['SH2ADD.UW'] },
]}
....

Description::
This instruction takes the least significant word of _rs1_, zero extends it, shifts it left by 2 places, and then adds it to _rs2_.

Operation::
[source,sail]
--

X(rd) = EXTZ(X(rs1)[31..0] + (X(rs2) << 2);
--
