== Count set bits
:page-partial:

Mnemonic::
cpop _rd_, _rs_
Encoding::
[wavedrom]
....
{reg:[
    { bits:  7, name: 0x13, attr: ['OP-IMM'] },
    { bits:  5, name: 'rd' },
    { bits:  3, name: 0x1, attr: ['SLLI'] },
    { bits:  5, name: 'rs' },
    { bits:  5, name: 0x2 },
    { bits:  7, name: 0x30 },
]}
....
Description:: 
Stores the number of set bits in _rs_ into _rd_.
Operation::
[source,sail]
--
let bitcount = 0;
let rs = X(rs);

foreach (i from 0 to (xlen - 1) in inc)
    if rs[i] == 0b1 then bitcount = bitcount + 1 else ();

X[rd] = bitcount
--
