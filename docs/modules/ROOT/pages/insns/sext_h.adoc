== Sign-extend halfword (16-bit)

Mnemonic::
sext.h _rd_, _rs_

Encoding::
[wavedrom]
....
{reg:[
    { bits:  7, name: 0x13, attr: ['OP-IMM'] },
    { bits:  5, name: 'rd' },
    { bits:  3, name: 0x1, attr: ['SEXT.B/SEXT.H'] },
    { bits:  5, name: 'rs1' },
    { bits:  5, name: 0x05, attr: ['SEXT.H'] },
    { bits:  7, name: 0x30 },
]}
....

Description::
Sign-extend the least-significant 16 bits of _rs_ to XLEN and place in _rd_.

Operation::
[source,sail]
--
X(rd) = EXTS(X(rs)[15..0]);
--
