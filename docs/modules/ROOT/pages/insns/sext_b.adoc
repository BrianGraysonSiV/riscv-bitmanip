== Sign-extend byte (8-bit)

Mnemonic::
sext.b _rd_, _rs_

Encoding::
[wavedrom]
....
{reg:[
    { bits:  7, name: 0x13, attr: ['OP-IMM'] },
    { bits:  5, name: 'rd' },
    { bits:  3, name: 0x1, attr: ['SEXT.B/SEXT.H'] },
    { bits:  5, name: 'rs1' },
    { bits:  5, name: 0x04, attr: ['SEXT.B'] },
    { bits:  7, name: 0x30 },
]}
....

Description::
Sign-extend the least-significant 8 bits of _rs_ to XLEN and place in _rd_.

Operation::
[source,sail]
--
X(rd) = EXTS(X(rs)[7..0]);
--
